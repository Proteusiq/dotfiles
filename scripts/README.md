## Module Responsibilities

### logging.sh
Core utilities for output and command execution.

| Function | Purpose |
|----------|---------|
| `log_step()` | Always-visible step headers (green) |
| `log()` | Verbose output with timestamp |
| `log_info()` | Info messages (blue, verbose only) |
| `log_warn()` | Warning messages (yellow, always) |
| `log_error()` | Error messages (red, always) |
| `log_debug()` | Debug output (-vv mode only) |
| `start_spinner()` | Show progress spinner |
| `stop_spinner()` | Hide spinner |
| `run_quiet()` | Run command with spinner (quiet) or full output (verbose) |
| `execute()` | Run command with debug logging |
| `has_cmd()` | Check if command exists |

**Verbosity Levels:**
- `0` (default): Only steps, warnings, errors
- `1` (-v): Detailed output with timestamps
- `2` (-vv): Debug mode, shows all executed commands

### ui.sh
Visual formatting for tables and output.

| Function | Purpose |
|----------|---------|
| `draw_table()` | Generic table with headers, widths, rows |
| `print_section()` | Section with icon, title, tool/version rows |

### versions.sh
Tool version management and queries.

| Function | CLI Flag | Purpose |
|----------|----------|---------|
| `show_installed_versions()` | `--versions [group]` | Display all installed tool versions |
| `get_tool_info()` | `--info <tool>` | Show detailed info about a tool |
| `show_outdated()` | `--outdated` | Check for available updates |
| `update_all()` | `--all` | Update all package managers |
| `update_tool()` | `update <tool>` | Update a single tool |
| `get_version()` | - | Get current version of a tracked tool |
| `track_version()` | - | Record version changes for summary |

## CLI Commands Reference

### Full Install
```bash
update                    # Quiet mode (spinner, minimal output)
update -v                 # Verbose (timestamps, details)
update -vv                # Debug (shows all commands)
update --dry-run          # Preview without executing
```

### Selective Install
```bash
update --only brew        # Run only install_brew function
update --only stow        # Run only stow_dotfiles function
update --list             # List all available functions
```

### Tool Updates
```bash
update bat                # Update single tool (auto-detects: brew)
update ruff               # Update single tool (auto-detects: uv)
update --all              # Update everything (brew, uv, cargo, git)
update --outdated         # Show what needs updating
```

### Information
```bash
update --versions         # Show all installed tools
update --versions brew    # Show only Homebrew formulae
update --versions uv      # Show only UV tools
update --info bat         # Show tool details, examples, category
update --help             # Show help
```

### Version Groups
| Group | Contents |
|-------|----------|
| `brew` | Homebrew formulae |
| `cask` | Homebrew casks (GUI apps) |
| `uv` | Python tools (uv tool install) |
| `cargo` | Rust packages |
| `llm` | LLM plugins |
| `git` | Git-cloned tools (tpm, yazi-flavors) |
| `other` | Standalone installs (bun, n, goose) |

## Adding New Tools

### To Brewfile (recommended)
```ruby
# In Brewfile
brew "newtool"        # CLI tool
cask "newapp"         # GUI application
```

### To setup_utils (special installation)
```bash
# In install.sh â†’ setup_utils()

    run_quiet some_install_command
```

```bash

    # ...existing...
    "newtool|newtool|cmd"  # name|version_cmd|type
```
